# misspell-en_UK

misspell-en_UK is an espanso package which is replacing american english with british english.
The package is based on [github.com/golangci/misspell](https://github.com/golangci/misspell).

## Installation

```
espanso install misspell-en_UK
espanso restart
```

## Usage

Type `color` and see what's happening.
