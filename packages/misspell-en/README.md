# misspell-en

misspell-en is an espanso package which is replacing commonly misspelled english words.
The package is based on [github.com/golangci/misspell](https://github.com/golangci/misspell).

## Installation

```
espanso install misspell-en
espanso restart
```

## Usage

Type `yuo` and see what's happening.
